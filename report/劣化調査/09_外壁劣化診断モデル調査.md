# 外壁劣化診断・シーリング検出モデル調査レポート

## 1. 概要

本レポートは、Python ベースのオープンソース（OSS）画像診断モデルに関する調査結果をまとめたものです。
主に以下の 2 つのテーマを調査・分析しました：

1. **外壁劣化診断モデル**：コンクリートひび割れ、腐食、クラックなどの検出
2. **シーリング（目地）検出**：外壁のシーリング材の自動検出・測定

---

## 2. 外壁劣化診断モデルの調査結果

### 2.1 主要な画像認識技術

| 技術 | 説明 | 適用事例 |
|------|------|---------|
| **セマンティックセグメンテーション** | ピクセル単位のクラス分類 | 劣化区域の細かな検出（ひび割れ、錆び） |
| **インスタンスセグメンテーション** | 個々の物体の境界検出 | 複数クラックの識別・計測 |

| **パノプティックセグメンテーション** | セマンティック + インスタンスの統合 | 総合的な劣化マッピング |

## 3. シーリング（目地）検出に関する調査

### 3.1 シーリング検出の技術的課題

| 課題 | 説明 | 対策案 |
|------|------|--------|
| **コントラストが低い** | 外壁とシーリングの色差が少ない | 色彩特徴量（HSV）の活用、データ拡張 |
| **材料の違い** | シリコン、ポリウレタン、アクリルなど | 多クラス分類（材料種類別） |
| **経年劣化** | 色褪せ、変色、劣化による特徴変化 | 多様なサンプルでの学習 |
| **角度変化** | 撮影角度による見え方の差異 | 多角的な画像データ収集 |

### 3.2 検出アプローチ

#### 3.2.1 セマンティックセグメンテーションベース

```
画像 → U-Net/DeepLab → 各ピクセルの分類（外壁，シーリング，ひび割れ）
```

**特徴**：
- 全ピクセルの分類が可能
- 劣化区域の正確な境界取得
- 面積計算が容易

#### 3.2.2 物体検出ベース

```
画像 → Mask R-CNN/YOLO → シーリング領域のボックス・マスク生成
```

**特徴**：
- 個々のシーリングラインの識別
- 個数カウント
- 複雑な配置への対応

#### 3.2.3 ハイブリッドアプローチ

```
画像 → 事前モデル → 転移学習 → 多クラスセグメンテーション
```

**クラス例**：
- 健全な外壁
- 健全なシーリング
- ひび割れ
- シーリングの劣化（割れ、剥がれ）
- 防水材の侵入

### 3.3 実装の推奨構成

#### 3.3.1 モデルアーキテクチャ

```
入力画像 (RGB)
  ↓
エンコーダ（ResNet50/101/ EfficientNet）
  ↓
デコーダ（U-Net/FPN/ASPP）
  ↓
出力（各ピクセルのクラス確率マップ）
```

#### 3.3.2 フレームワーク選択

| 要件 | 推奨 |
|------|------|
| **転移学習重視** | torchvision + 学習済みモデル |
| **柔軟なカスタマイズ** | PyTorch + segmentation_models_pytorch |
| **高速推論** | YOLOv8 (ultralytics) |
| **大規模モデル** | Detectron2 |

#### 3.3.3 推奨実装フロー

1. **データ収集**
   - 外壁画像の収集（各種劣化状態）
   - ラベル付け（LabelImg, CVAT など）

2. **モデル選定**
   - U-Net または DeepLab V3+（セグメンテーション）
   - 転移学習済みモデルからの始動

3. **学習・検証**
   - 多クラスセグメンテーション
   - 交差エントロピー・IoU 最適化

4. **デプロイ**
   - ONNX または TorchScript による高速化
   - API 化（Flask/FastAPI）

## 4. Python 実装例

### 4.1 セマンティックセグメンテーション（U-Net）

```python
import torch
from segmentation_models_pytorch import Unet
from segmentation_models_pytorch.utils.losses import DiceLoss

# モデル作成
model = Unet(
    encoder_name='resnet34',
    encoder_weights='imagenet',
    num_classes=3,  # 外壁，シーリング，ひび割れ
    in_channels=3
)

# ロス関数
criterion = DiceLoss(num_classes=3)
optimizer = torch.optim.Adam(model.parameters(), lr=1e-4)
```

### 4.2 転移学習済み Mask R-CNN の利用

```python
from torchvision.models.detection.mask_rcnn import MaskRCNN

model = torch.hub.load('facebookresearch/detectron2', 'maskrcnn_resnet50_fpn', pretrained=True)
model.eval()

images, _ = model.image_from_file('crack.jpg')
preds = model(images)
```

### 4.3 Ultralytics YOLOv8 セグメンテーション

```python
from ultralytics import SegmentationModel

model = SegmentationModel.from_pretrained('yolo-seg8x-imagenet.pt')
results = model('wall_crack.jpg')
```

## 6. まとめ・推奨アクション

### 6.1 調査の結論

1. **OSS で利用可能なモデルが豊富**
   - PyTorch/TensorFlow で多数の実装例
   - 転移学習済みモデルの利用で初期実装が容易

2. **セマンティックセグメンテーションが推奨**
   - ピクセル単位の劣化区域検出に適
   - U-Net, DeepLab V3+ が標準的

3. **シーリング検出は多クラスセグメンテーションで可能**
   - 外壁とシーリングの両方を同時に分類
   - 転移学習で効率よく学習可能
